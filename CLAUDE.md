# CLAUDE.md

- Roadmap is tracked in `TODO`.
- Bugs are tracked in `BUGS`.
- Both are plain lowercase text files; entries begin with a dash (`-`).
- Newly discovered bugs are appended to the end of `BUGS`.
- `BUGS` entries follow this format:

  ```
  - <canonical-name>: <description>

      <JS code that triggers it>

  ```
